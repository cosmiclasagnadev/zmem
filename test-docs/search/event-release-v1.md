# Release v1.0.0

We are excited to announce the release of version 1.0.0 of our platform!

## What's new

- Core memory system with hybrid search
- MCP server integration for OpenCode
- SQLite storage with FTS5 full-text search
- Vector embeddings with zvec

## Migration notes

Users upgrading from beta versions should run the migration script to ensure data integrity.

Special thanks to all contributors who made this release possible.
